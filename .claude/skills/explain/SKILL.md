# Explain Mode

Break down a concept, term, or legal principle from COUN 5553 into plain language.

## How It Works

1. If the user specifies a concept — read relevant materials and explain it
2. If no concept is specified, ask what they want explained
3. Structure every explanation as:
   - **Formal definition** — the textbook/legal definition
   - **Plain English** — what it actually means in everyday language
   - **Clinical example** — how this shows up in real counseling practice
   - **Why it matters** — why a counselor needs to know this
4. If the concept connects to a landmark case, mention it briefly
5. If it relates to a specific ACA Code section, cite it

## Guidelines

- Read the relevant course materials before explaining — use the course's framing, not generic definitions
- Avoid oversimplifying legal nuances — explain the complexity, but make it accessible
- If a term has multiple meanings in different contexts (legal vs clinical vs everyday), clarify all of them
- After explaining, ask if it makes sense or if they want to go deeper on any part
