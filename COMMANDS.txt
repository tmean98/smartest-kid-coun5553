SMARTEST KID — TERMINAL COMMANDS CHEATSHEET
===========================================

## Start the dev server
cd "/Users/thomasmeaney/Coding Praccy/smartest-kid-coun5553/web" && npm run dev
→ Opens at http://localhost:3000

## Stop the dev server
Ctrl+C in the terminal where it's running

## Install dependencies (if something breaks after a pull)
cd "/Users/thomasmeaney/Coding Praccy/smartest-kid-coun5553/web" && npm install

## Run the LMS sync script (downloads new materials from OpenLMS)
cd "/Users/thomasmeaney/Coding Praccy/smartest-kid-coun5553" && node sync.mjs

## Rebuild course chunks (after new materials are added)
cd "/Users/thomasmeaney/Coding Praccy/smartest-kid-coun5553/web" && npx ts-node scripts/build-chunks.ts --all
# Or for a single course:
cd "/Users/thomasmeaney/Coding Praccy/smartest-kid-coun5553/web" && npx ts-node scripts/build-chunks.ts --course coun5553

## Git — check what's changed
cd "/Users/thomasmeaney/Coding Praccy/smartest-kid-coun5553" && git status

## Git — pull latest changes
cd "/Users/thomasmeaney/Coding Praccy/smartest-kid-coun5553" && git pull
